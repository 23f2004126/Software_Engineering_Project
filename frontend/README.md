# Sonik Smart Retail Management System - Frontend

A modern, responsive Vue 3 web application for comprehensive retail store management including inventory, billing, supplier management, employee shifts, credit management, and milk subscriptions.

## Features

For Store Owners
-Dashboard - Real-time KPIs, revenue charts, and AI-powered insights
-Inventory Management - Track stock levels, expiry dates, damage/loss reports
-Billing & Sales - Point-of-sale, sales history, detailed transaction reports
-Credit Management - Customer credit limits, payment tracking, risk assessment
-Supplier Management - Supplier profiles, purchase orders, payment tracking
-Finance & Reports - Financial analytics, sales reports, damage reports
-Employee Management - Staff profiles and performance tracking
-Milk Subscriptions - Manage daily milk delivery subscriptions
-AI Chat Assistant - Real-time insights and recommendations

For Store Employees
-Shift Login/Logout - Track daily shift timings
-Limited Dashboard - Today's sales and low stock alerts
-Inventory Access - View inventory and log damage/losses
-Shift Reports - Daily sales and performance summary
-Milk Daily Entry - Process daily milk deliveries
-Billing - Point-of-sale transactions

## Prerequisites

- Node.js- v16 or higher
- npm- v7 or higher (or yarn)

## Installation

### Install Dependencies
```bash
npm ci
```

This will install all required packages including:
- Vue 3 - Progressive JavaScript framework
- Vue Router 5 - Client-side routing
- Pinia - State management
- Tailwind CSS - Utility-first CSS framework
- Lucide Vue Next - Beautiful UI icons
- And other essential packages

## Running the Application

### Development Server
```bash
npm run dev
```

The application will start on `http://localhost:5173` (or next available port shown in terminal).
